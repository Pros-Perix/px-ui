## Installation

Install PX-UI using your preferred package manager:

```bash title="npm"
npm install @px-ui/core
```

```bash title="pnpm"
pnpm add @px-ui/core
```

```bash title="yarn"
yarn add @px-ui/core
```

## Setup

### 1. Import Styles

Import the PX-UI styles in your main CSS file or entry point:

```css title="app.css"
@import "@px-ui/core/styles";
```

Or import in your JavaScript/TypeScript entry point:

```tsx title="main.tsx"
import '@px-ui/core/styles';
```

### 2. Configure Tailwind (Optional)

If you're using Tailwind CSS in your project, you may want to extend your Tailwind configuration to include PX-UI's design tokens:

```js title="tailwind.config.js"
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    './src/**/*.{js,ts,jsx,tsx}',
    './node_modules/@px-ui/core/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

## Usage

Import and use components in your React application:

```tsx title="app.tsx"
import { Button, Input, Select } from '@px-ui/core';

function App() {
  return (
    <div>
      <h1>My App</h1>
      <Button variant="primary">Get Started</Button>
      <Input placeholder="Enter your email" />
    </div>
  );
}
```

## Monorepo Setup

If you're working in a monorepo (like this project), you can reference the package using workspace protocol:

```json title="package.json"
{
  "dependencies": {
    "@px-ui/core": "workspace:*"
  }
}
```

## Tree Shaking

PX-UI is built with tree-shaking in mind. You can import individual components to reduce bundle size:

```tsx
// Import only what you need
import { Button } from '@px-ui/core/button';
import { Input } from '@px-ui/core/input';
```

Or import from the main entry:

```tsx
// Bundle tools will tree-shake unused exports
import { Button, Input } from '@px-ui/core';
```

## TypeScript

PX-UI is written in TypeScript and provides full type definitions out of the box. No additional setup is required.

```tsx
import type { ButtonProps } from '@px-ui/core';

const MyButton: React.FC<ButtonProps> = (props) => {
  return <Button {...props} />;
};
```
