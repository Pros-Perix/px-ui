## Overview

The Label component provides accessible labeling for form controls. It automatically handles disabled states and integrates with form inputs.

## Import

```tsx
import { Label } from "@px-ui/core";
```

## Usage

<Preview>
  <div className="flex flex-col gap-2">
    <Label htmlFor="email">Email</Label>
    <input id="email" type="email" placeholder="Enter your email" className="px-3 py-2 border rounded-md" />
  </div>
</Preview>

<CodeBlock>
```tsx
<Label htmlFor="email">Email</Label>
<input id="email" type="email" placeholder="Enter your email" />
```
</CodeBlock>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | - | Associates the label with a form control |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Label content |

Plus all standard HTML label attributes.
