## Overview

The Avatar component displays user profile pictures with automatic fallback to colored initials when an image is not available. It's built on top of Base UI's Avatar component and includes tooltip support.

## Import

```tsx
import { Avatar } from "@px-ui/core";
```

<Preview>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `imgSrc` | `string \| null \| undefined` | - | Image source URL for the avatar |
| `name` | `string \| null \| undefined` | - | Name used for initials fallback and tooltip |
| `variant` | `"squared" \| "rounded"` | `"squared"` | Visual style variant |
| `size` | `` `${number}px` `` | `"100px"` | Size of the avatar in pixels |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Additional content to render inside the avatar |
| `hideTooltip` | `boolean` | `false` | Hide the tooltip that shows the name |
