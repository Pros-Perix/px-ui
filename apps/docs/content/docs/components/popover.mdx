## Overview

The Popover component displays content in a floating overlay anchored to a trigger element. It's built on top of Base UI's Popover component and is perfect for tooltips, dropdowns, and contextual information.

## Import

```tsx
import { Popover } from "@px-ui/core";
```

## Usage

<Preview>
  <Popover.Root>
    <Popover.Trigger render={<Button variant="outline">Open Popover</Button>} />
    <Popover.Content>
      <Popover.Header>
        <Popover.Title>Popover Title</Popover.Title>
        <Popover.Description>
          This is a popover description.

        <p>Popover content goes here.</p>
      </div>
    </Popover.Content>

</Preview>

## Props

### Popover.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `defaultOpen` | `boolean` | - | Uncontrolled default open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `openOnHover` | `boolean` | `false` | Open on hover instead of click |

### Popover.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `render` | `ReactNode \| ComponentProps` | - | Custom trigger element or render props |
| `disabled` | `boolean` | - | Disable the trigger |

### Popover.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `arrow` | `boolean` | `true` | Show arrow pointing to trigger |
| `positionerProps` | `object` | - | Props for Popover.Positioner (side, align, sideOffset) |
| `popupProps` | `object` | - | Props for Popover.Popup |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Popover content |

### Popover.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Header content |

### Popover.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Title text |

### Popover.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Description text |

### Popover.Footer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Footer content |

### Popover.Close

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `render` | `ReactNode \| ComponentProps` | - | Custom close element or render props |

Plus all standard Base UI Popover props.
