## Overview

Xandi is an AI chatbot component built on top of Vercel's AI SDK (`@ai-sdk/react`). It provides a simple, ready-to-use chat interface that connects to your AI backend.

## Import

```tsx
import { Xandi } from "@px-ui/ai";
```

## Backend Setup

Xandi expects an API endpoint that follows the Vercel AI SDK streaming protocol. Here's an example using Next.js:

<CodeBlock>
```tsx
// app/api/chat/route.ts
import { openai } from "@ai-sdk/openai";
import { streamText } from "ai";

export async function POST(req: Request) {
  const { messages } = await req.json();

  const result = streamText({
    model: openai("gpt-4"),
    messages,
  });

  return result.toDataStreamResponse();
}
```
</CodeBlock>

## How It Works

Xandi uses the `useChat` hook from `@ai-sdk/react` with `DefaultChatTransport` to handle:

- **Message state management** - Automatically manages the conversation history
- **Streaming responses** - Displays AI responses as they stream in
- **Connection status** - Tracks status (`ready`, `loading`, etc.) to disable input during generation

## API Reference

### Xandi

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `api` | `string` | `"/api/chat"` | The API endpoint for the chat backend |

| `ai` | >= 6.0.0 |
