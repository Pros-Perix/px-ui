## Overview

The RadioGroup component provides a group of radio buttons for selecting a single option from multiple choices. It's built on top of Base UI's RadioGroup component and supports multiple sizes.

## Import

```tsx
import { RadioGroup } from "@px-ui/core";
```

## Props

### RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled selected value |
| `defaultValue` | `string` | - | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `disabled` | `boolean` | - | Disable all radio items |
| `className` | `string` | - | Additional CSS classes |

### RadioGroup.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Value of the radio item |
| `disabled` | `boolean` | - | Disable the radio item |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the radio button |
| `variant` | `"default"` | `"default"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |

Plus all standard Base UI RadioGroup and Radio props.
