## Overview

The RadioGroup component provides a set of radio buttons where only one option can be selected at a time.

## Installation

```tsx
import * as RadioGroup from "@px-ui/core";
```

## Base-UI Reference

This component is built on top of [Base UI RadioGroup](https://base-ui.com/react/components/radio-group).

## Usage

### Basic Usage

```tsx
import * as RadioGroup from "@px-ui/core";

<RadioGroup.Group>
  <div className="flex items-center gap-2">
    <RadioGroup.Item value="option1" />
    <label>Option 1</label>
  </div>
  <div className="flex items-center gap-2">
    <RadioGroup.Item value="option2" />
    <label>Option 2</label>
  </div>
  <div className="flex items-center gap-2">
    <RadioGroup.Item value="option3" />
    <label>Option 3</label>
  </div>
</RadioGroup.Group>
```

### Controlled State

```tsx
import { useState } from "react";

function Example() {
  const [value, setValue] = useState("option1");

  return (
    <RadioGroup.Group value={value} onValueChange={setValue}>
      <div className="flex items-center gap-2">
        <RadioGroup.Item value="option1" />
        <label>Option 1</label>
      </div>
      <div className="flex items-center gap-2">
        <RadioGroup.Item value="option2" />
        <label>Option 2</label>
      </div>
    </RadioGroup.Group>
  );
}
```

### Sizes

```tsx
<RadioGroup.Group>
  <RadioGroup.Item value="sm" size="sm" />   // 16px
  <RadioGroup.Item value="default" />         // 20px (default)
  <RadioGroup.Item value="lg" size="lg" />   // 24px
</RadioGroup.Group>
```

### Disabled State

```tsx
<RadioGroup.Group disabled>
  <div className="flex items-center gap-2">
    <RadioGroup.Item value="disabled" />
    <label>Disabled option</label>
  </div>
</RadioGroup.Group>
```

## Components

### RadioGroup.Group

The container component that manages the selection state.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled selected value |
| `defaultValue` | `string` | - | Initial selected value (uncontrolled) |
| `onValueChange` | `(value: string) => void` | - | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disable all radio items |
| `name` | `string` | - | Form field name |
| `className` | `string` | - | Additional CSS classes |

### RadioGroup.Item

Individual radio button.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Value of this option |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the radio button |
| `disabled` | `boolean` | `false` | Disable this specific item |
| `className` | `string` | - | Additional CSS classes |
