## Overview

The Dialog component displays modal content in an overlay. It's built on top of Base UI's Dialog component and supports nested dialogs, custom headers with icons, and flexible content layouts.

## Import

```tsx
import { Dialog } from "@px-ui/core";
```

## Usage

<Preview>
  <Dialog.Root>
    <Dialog.Trigger render={<Button>Open Dialog</Button>} />
    <Dialog.Content>
      <Dialog.Header>
        <Dialog.HeaderContent>
          <Dialog.Title>Dialog Title</Dialog.Title>
          <Dialog.Description>
            This is a dialog description.
          </Dialog.Description>
        </Dialog.HeaderContent>
      </Dialog.Header>
      <div className="space-y-4">
        <p>Dialog content goes here.</p>
      </div>
      <Dialog.Footer>
        <Dialog.Close render={<Button variant="outline">Cancel</Button>} />
        <Button>Confirm</Button>

    </Dialog.Content>
  </Dialog.Root>

## Props

### Dialog.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `defaultOpen` | `boolean` | - | Uncontrolled default open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `modal` | `boolean` | `true` | Whether the dialog is modal |

### Dialog.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `render` | `ReactNode \| ComponentProps` | - | Custom trigger element or render props |

### Dialog.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Dialog content |

### Dialog.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Header content |

### Dialog.HeaderIcon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Icon content |

### Dialog.HeaderContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Header content (Title and Description) |

### Dialog.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Title text |

### Dialog.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Description text |

### Dialog.Footer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Footer content |

### Dialog.Close

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `render` | `ReactNode \| ComponentProps` | - | Custom close element or render props |

Plus all standard Base UI Dialog props.
