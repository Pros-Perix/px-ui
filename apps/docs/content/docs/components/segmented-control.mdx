## Overview

The SegmentedControl component displays a group of options as segments in a single control. It's built on top of Base UI's RadioGroup component and provides a modern, iOS-style segmented control interface.

## Import

```tsx
import { SegmentedControl } from "@px-ui/core";
```

## Props

### SegmentedControl.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled selected value |
| `defaultValue` | `string` | - | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `disabled` | `boolean` | - | Disable all items |
| `className` | `string` | - | Additional CSS classes |

### SegmentedControl.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Item value |
| `disabled` | `boolean` | - | Disable the item |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the item |
| `variant` | `"default"` | `"default"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content |

Plus all standard Base UI RadioGroup props.
