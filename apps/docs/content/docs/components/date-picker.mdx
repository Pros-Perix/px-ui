## Overview

The DatePicker component provides a date selection interface by combining the Calendar component with a Popover trigger. It supports single date, multiple dates, and date range selection modes.

## Import

```tsx
import { DatePicker } from "@px-ui/core";
```

## Usage

<Preview>
  <DatePicker
    mode="single"
    selected={new Date()}
    onSelect={(date) => console.log(date)}
  />
</Preview>

<CodeBlock>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `"single" \| "multiple" \| "range"` | `"single"` | Selection mode (from Calendar) |
| `selected` | `Date \| Date[] \| { from: Date; to?: Date }` | - | Selected date(s) |
| `onSelect` | `(date: Date \| Date[] \| { from: Date; to?: Date }) => void` | - | Callback when date is selected |
| `placeholder` | `string` | `"Pick a date"` | Placeholder text |
| `format` | `string` | `"yyyy-MM-dd"` | Date format string (date-fns format) |
| `invalid` | `boolean` | - | Invalid state |
| `triggerProps` | `object` | - | Props for the trigger button (size, widthVariant, disabled, className) |
| `trailingIcon` | `ReactNode` | - | Custom trailing icon |
| `renderTriggerContent` | `ReactNode` | - | Custom trigger content |
| `popoverRootProps` | `object` | - | Props for Popover.Root |
| `popoverContentProps` | `object` | - | Props for Popover.Content |
| `className` | `string` | - | Additional CSS classes |

Plus all Calendar component props (captionLayout, numberOfMonths, showWeekNumber, disabled, fromYear, toYear, etc.).
