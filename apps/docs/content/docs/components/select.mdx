## Overview

The Select component provides a dropdown interface for selecting a value from a list of options. It's built on top of Base UI's Select component and supports single and multiple selection modes.

## Import

```tsx
import { Select } from "@px-ui/core";
```

## Props

### Select.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `any \| any[]` | - | Selected value(s) |
| `defaultValue` | `any \| any[]` | - | Default selected value(s) |
| `onValueChange` | `(value: any \| any[]) => void` | - | Callback when value changes |
| `multiple` | `boolean` | `false` | Enable multiple selection |
| `invalid` | `boolean` | - | Invalid state |
| `disabled` | `boolean` | - | Disable the select |
| `isItemEqualToValue` | `(item: any, value: any) => boolean` | - | Custom equality function |

### Select.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | - | Size of the trigger |
| `widthVariant` | `"enforced" \| "full"` | - | Width behavior |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Trigger content (typically Value) |

### Select.Value

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text |

### Select.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `widthVariant` | `"trigger" \| "fit" \| "enforced"` | `"trigger"` | Width behavior |
| `portalProps` | `object` | - | Props for Select.Portal |
| `positionerProps` | `object` | - | Props for Select.Positioner |
| `popupProps` | `object` | - | Props for Select.Popup |
| `children` | `ReactNode` | - | Content (typically List) |

### Select.List

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | List items |

### Select.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `any` | - | Item value |
| `disabled` | `boolean` | - | Disable the item |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content |

### Select.MultiItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `any` | - | Item value |
| `disabled` | `boolean` | - | Disable the item |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content |

Plus all standard Base UI Select props.
