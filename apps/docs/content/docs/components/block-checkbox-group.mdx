## Overview

The BlockCheckboxGroup component displays checkboxes in card-style blocks, perfect for selecting multiple options with rich content like titles and descriptions. It's built on top of Base UI's CheckboxGroup component.

## Import

```tsx
import { BlockCheckboxGroup } from "@px-ui/core";
```

## Props

### BlockCheckboxGroup.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `allValues` | `string[]` | - | All possible values in the group |
| `value` | `string[]` | - | Controlled selected values |
| `defaultValue` | `string[]` | - | Uncontrolled default selected values |
| `onValueChange` | `(values: string[]) => void` | - | Callback when values change |
| `disabled` | `boolean` | - | Disable all items |
| `className` | `string` | - | Additional CSS classes |

### BlockCheckboxGroup.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Value of the checkbox item |
| `parent` | `boolean` | - | Mark as parent checkbox (for select all) |
| `indeterminate` | `boolean` | - | Indeterminate state |
| `invalid` | `boolean` | - | Invalid state |
| `disabled` | `boolean` | - | Disable the item |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content (typically Header and Description) |

### BlockCheckboxGroup.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Header content |

### BlockCheckboxGroup.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Title text |

### BlockCheckboxGroup.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Description text |

Plus all standard Base UI CheckboxGroup props.
