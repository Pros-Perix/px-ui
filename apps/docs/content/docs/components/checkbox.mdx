## Overview

The Checkbox component is a styled checkbox input built on top of Base UI's Checkbox component. It supports multiple sizes and can be used in controlled or uncontrolled forms.

## Import

```tsx
import { Checkbox } from "@px-ui/core";
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | - | Controlled checked state |
| `defaultChecked` | `boolean` | - | Uncontrolled default checked state |
| `indeterminate` | `boolean` | - | Indeterminate state |
| `disabled` | `boolean` | - | Disable the checkbox |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the checkbox |
| `variant` | `"default"` | `"default"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |
| `onCheckedChange` | `(checked: boolean) => void` | - | Callback when checked state changes |

Plus all standard Base UI Checkbox props.
