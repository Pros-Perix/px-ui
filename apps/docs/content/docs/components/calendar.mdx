## Overview

The Calendar component provides a flexible date picker interface supporting single date selection, multiple dates, and date ranges. It's built on top of react-day-picker and provides a styled, accessible calendar interface.

## Import

```tsx
import { Calendar } from "@px-ui/core";
```

## Usage

<Preview>

    mode="single"
    selected={new Date()}
    onSelect={(date) => console.log(date)}

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `"single" \| "multiple" \| "range"` | `"single"` | Selection mode |
| `selected` | `Date \| Date[] \| { from: Date; to?: Date }` | - | Selected date(s) |
| `onSelect` | `(date: Date \| Date[] \| { from: Date; to?: Date }) => void` | - | Callback when date is selected |
| `defaultMonth` | `Date` | `new Date()` | Default month to display |
| `month` | `Date` | - | Controlled month |
| `onMonthChange` | `(month: Date) => void` | - | Callback when month changes |
| `captionLayout` | `"label" \| "dropdown" \| "dropdown-months" \| "dropdown-years"` | `"label"` | Caption layout style |
| `numberOfMonths` | `number` | `1` | Number of months to display |
| `showOutsideDays` | `boolean` | `true` | Show days from previous/next month |
| `showWeekNumber` | `boolean` | `false` | Show week numbers |
| `disabled` | `Date \| Date[] \| { before?: Date; after?: Date }` | - | Disabled dates |
| `fromYear` | `number` | - | Start year for dropdown |
| `toYear` | `number` | - | End year for dropdown |
| `className` | `string` | - | Additional CSS classes |

Plus all standard react-day-picker DayPickerProps.
