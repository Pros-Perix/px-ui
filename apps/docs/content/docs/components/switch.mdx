## Overview

The Switch component is a toggle control that allows users to switch between two states. It's built on top of Base UI's Switch component and supports multiple sizes.

## Import

```tsx
import { Switch } from "@px-ui/core";
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | - | Controlled checked state |
| `defaultChecked` | `boolean` | - | Uncontrolled default checked state |
| `disabled` | `boolean` | - | Disable the switch |
| `size` | `"sm" \| "lg"` | - | Size of the switch (default is medium) |
| `className` | `string` | - | Additional CSS classes |
| `onCheckedChange` | `(checked: boolean) => void` | - | Callback when checked state changes |

Plus all standard Base UI Switch props.
