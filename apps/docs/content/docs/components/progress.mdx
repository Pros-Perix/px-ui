## Overview

The Progress component displays the completion status of a task. It's built on top of Base UI's Progress component and supports multiple sizes, labels, and value formatting.

## Import

```tsx
import { Progress } from "@px-ui/core";
```

## Usage

<Preview>
  <Progress.Root value={75} max={100}>
    <div className="flex items-center justify-between mb-2">
      <Progress.Label>Progress</Progress.Label>
      <Progress.Value />
    </div>
    <Progress.Track>

## Props

### Progress.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Current progress value |
| `max` | `number` | `100` | Maximum value |
| `format` | `Intl.NumberFormatOptions` | - | Format options for the value display |
| `className` | `string` | - | Additional CSS classes |

### Progress.Track

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the progress track |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Track content (typically Indicator) |

### Progress.Indicator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### Progress.Label

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Label text |

### Progress.Value

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

Plus all standard Base UI Progress props.
