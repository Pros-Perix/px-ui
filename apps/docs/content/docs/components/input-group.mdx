## Overview

The InputGroup component allows you to combine input fields with addons (icons, text, or buttons) on either side. It's perfect for creating inputs with prefixes, suffixes, or action buttons.

## Import

```tsx
import { InputGroup } from "@px-ui/core";
```

## Usage

<Preview>
  <InputGroup.Root>
    <InputGroup.Input placeholder="Search..." />
    <InputGroup.Addon>
      <svg width="16" height="16" viewBox="0 0 16 16" fill="none" stroke="currentColor" strokeWidth="2">
        <circle cx="7" cy="7" r="4" />
        <path d="m10 10 3 3" />
      </svg>
    </InputGroup.Addon>
  </InputGroup.Root>
</Preview>

<CodeBlock>
```tsx

  <InputGroup.Input placeholder="Search..." />
```

## Props

### InputGroup.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Size of the input group |
| `widthVariant` | `"enforced" \| "fit" \| "full"` | `"full"` | Width behavior |
| `disabled` | `boolean` | - | Disable the entire input group |
| `className` | `string` | - | Additional CSS classes |

### InputGroup.Input

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `invalid` | `boolean` | - | Invalid state |
| `type` | `string` | `"text"` | Input type |
| `placeholder` | `string` | - | Placeholder text |
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML input attributes.

### InputGroup.Addon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"inline-start" \| "inline-end"` | `"inline-start"` | Alignment of the addon |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Addon content (icon, text, or button) |

### InputGroup.Text

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Text content |

### InputGroup.Button

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"xs" \| "sm" \| "icon-xs" \| "icon-sm"` | `"xs"` | Size of the button |
| `variant` | `ButtonVariant` | `"ghost"` | Button variant |
| `type` | `string` | `"button"` | Button type |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Button content |

Plus all standard HTML button attributes.
