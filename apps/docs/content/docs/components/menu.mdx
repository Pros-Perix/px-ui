## Overview

The Menu component displays a dropdown list of actions. It's built on top of Base UI's Menu component and supports separators, groups, and radio groups.

## Import

```tsx
import { Menu } from "@px-ui/core";
```

## Usage

<Preview>
  <Menu.Root>
    <Menu.Trigger>File</Menu.Trigger>
    <Menu.Content>
      <Menu.Item onClick={() => console.log("New File")}>New File</Menu.Item>

## Props

### Menu.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | Controlled open state |
| `defaultOpen` | `boolean` | - | Uncontrolled default open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |

### Menu.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | - | Size of the trigger |
| `widthVariant` | `"enforced" \| "fit" \| "full"` | `"fit"` | Width behavior |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Trigger content |

### Menu.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `widthVariant` | `"trigger" \| "fit" \| "enforced"` | `"fit"` | Width behavior |
| `portalProps` | `object` | - | Props for Menu.Portal |
| `positionerProps` | `object` | - | Props for Menu.Positioner |
| `popupProps` | `object` | - | Props for Menu.Popup |
| `children` | `ReactNode` | - | Menu content |

### Menu.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onClick` | `() => void` | - | Click handler |
| `disabled` | `boolean` | - | Disable the item |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content |

### Menu.Separator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### Menu.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Group content |

### Menu.GroupLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Label text |

### Menu.RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `children` | `ReactNode` | - | Radio items |

### Menu.RadioItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Radio item value |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content |

Plus all standard Base UI Menu props.
