## Overview

The Input component is a styled text input field that supports multiple sizes, validation states, and width variants. It provides consistent styling and accessibility features.

## Import

```tsx
import { Input } from "@px-ui/core";
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Size of the input |
| `widthVariant` | `"enforced" \| "full"` | `"full"` | Width behavior of the input |
| `invalid` | `boolean` | - | Whether the input is in an invalid state |
| `disabled` | `boolean` | - | Whether the input is disabled |
| `type` | `string` | `"text"` | The type of input (text, email, password, number, search, tel, url) |
| `className` | `string` | - | Additional CSS classes |
| `placeholder` | `string` | - | Placeholder text |

Plus all standard HTML input attributes.
