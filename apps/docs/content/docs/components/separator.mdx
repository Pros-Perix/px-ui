## Overview

The Separator component provides a visual divider between content sections. It can be displayed horizontally or vertically and supports decorative mode for purely visual separators.

## Import

```tsx
import { Separator } from "@px-ui/core";
```

## Usage

<Preview>
  <div className="flex flex-col gap-4 w-full">
    <div>Content above</div>
    <Separator orientation="horizontal" />
    <div>Content below</div>
  </div>
</Preview>

<CodeBlock>

<div>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Orientation of the separator |
| `decorative` | `boolean` | `false` | If true, the separator is decorative and won't be announced by screen readers |
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML div attributes.
