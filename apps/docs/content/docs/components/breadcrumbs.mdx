## Overview

The Breadcrumbs component displays a navigation trail showing the user's current location within a site hierarchy. It uses Base UI utilities for flexible rendering.

## Import

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@px-ui/core";
```

## Props

### Breadcrumb

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML nav attributes.

### BreadcrumbList

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML ol attributes.

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML li attributes.

### BreadcrumbLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | Link URL |
| `render` | `ComponentProps` | - | Custom render function |
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML a attributes.

### BreadcrumbPage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

Plus all standard HTML span attributes.

### BreadcrumbSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Custom separator content |

Plus all standard HTML li attributes.
