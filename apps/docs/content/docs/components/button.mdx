## Overview

The Button component is a versatile, styled button that supports multiple visual variants and sizes. It's built using class-variance-authority for type-safe variant management.

## Import

```tsx
import { Button, buttonVariants } from "@px-ui/core";
```

## Using buttonVariants

The `buttonVariants` function can be used to apply button styles to other elements like links:

```tsx
import { buttonVariants } from "@px-ui/core";
import { Link } from "your-router";

<Link
  to="/dashboard"
  className={buttonVariants({ variant: "primary", size: "lg" })}
>
  Go to Dashboard
</Link>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "primary" \| "destructive" \| "outline" \| "primary-outline" \| "ghost" \| "link"` | `"default"` | Visual style variant |
| `size` | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm"` | `"default"` | Size of the button |
| `disabled` | `boolean` | `false` | Disable the button |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Button content |

Plus all standard HTML button attributes.
