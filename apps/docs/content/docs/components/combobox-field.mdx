## Overview

The ComboboxField component provides a simplified interface for the Combobox component, automatically handling label rendering, search functionality, and common use cases. It's perfect for forms where you need searchable selection from a list of items.

## Import

```tsx
import { ComboboxField } from "@px-ui/forms";
```

## Usage

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `ReadonlyArray<TItem>` | - | Array of items to display (not needed if using loadOptions) |
| `loadOptions` | `LoadOptionsConfig<TItem>` | - | Async options loader configuration |
| `value` | `TSelectedValue \| TSelectedValue[]` | - | Selected value(s) |
| `onValueChange` | `(value: TSelectedValue \| TSelectedValue[]) => void` | - | Callback when value changes |
| `renderLabel` | `(item: TSelectedValue) => string \| ReactNode` | - | Function to render the label in the trigger (auto-detects if item has 'label' property) |
| `renderOption` | `(item: TItem) => ReactNode` | - | Function to render each option in the dropdown (auto-detects if item has 'label' property) |
| `renderChip` | `(item: TSelectedValue) => ReactNode \| string` | - | Function to render each chip in ChipsTrigger (multiple selection only) |
| `placeholder` | `string` | - | Placeholder text when no value is selected |
| `searchInPopup` | `boolean` | `false` | Show search input inside the popup instead of in trigger |
| `size` | `"default" \| "sm"` | - | Size variant for trigger |
| `widthVariant` | `"enforced" \| "full"` | - | Width variant for trigger |
| `contentWidthVariant` | `"trigger" \| "fit" \| "enforced"` | - | Width variant for the dropdown content |
| `triggerClassName` | `string` | - | Additional CSS classes for the trigger |
| `contentProps` | `object` | - | Additional props for Combobox.Content |
| `multiple` | `boolean` | `false` | Enable multiple selection |
| `disabled` | `boolean` | - | Disable the combobox |
| `invalid` | `boolean` | - | Invalid state |
| `isItemEqualToValue` | `(item: TItem, value: TSelectedValue) => boolean` | - | Custom equality function for comparing items |

Plus all standard Combobox.Root props.
