## Overview

The BlockRadioGroup component displays radio buttons in card-style blocks, perfect for selecting a single option from multiple choices with rich content like titles and descriptions. It's built on top of Base UI's RadioGroup component.

## Import

```tsx
import { BlockRadioGroup } from "@px-ui/core";
```

## Props

### BlockRadioGroup.Group

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled selected value |
| `defaultValue` | `string` | - | Uncontrolled default selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `disabled` | `boolean` | - | Disable all items |
| `className` | `string` | - | Additional CSS classes |

### BlockRadioGroup.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Value of the radio item |
| `invalid` | `boolean` | - | Invalid state |
| `disabled` | `boolean` | - | Disable the item |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Item content (typically Header and Description) |

### BlockRadioGroup.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Header content |

### BlockRadioGroup.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Title text |

### BlockRadioGroup.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Description text |

Plus all standard Base UI RadioGroup props.
