## Overview

The SelectField component provides a simplified interface for the Select component, automatically handling label rendering and common use cases. It's perfect for forms where you have a simple list of items to select from.

## Import

```tsx
import { SelectField } from "@px-ui/forms";
```

## Usage

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `ReadonlyArray<TItem>` | - | Array of items to display in the select dropdown |
| `value` | `TItem \| TItem[]` | - | Selected value(s) |
| `onValueChange` | `(value: TItem \| TItem[]) => void` | - | Callback when value changes |
| `renderLabel` | `(item: TItem) => ReactNode` | - | Function to render the label in the trigger (auto-detects if item has 'label' property) |
| `renderOption` | `(item: TItem) => ReactNode` | - | Function to render each option in the dropdown (auto-detects if item has 'label' property) |
| `placeholder` | `string` | - | Placeholder text when no value is selected |
| `size` | `"default" \| "sm"` | - | Size of the trigger |
| `widthVariant` | `"enforced" \| "full"` | - | Width behavior of the trigger |
| `contentWidthVariant` | `"trigger" \| "fit" \| "enforced"` | - | Width variant for the dropdown content |
| `triggerClassName` | `string` | - | Additional CSS classes for the trigger |
| `contentProps` | `object` | - | Additional props for Select.Content |
| `multiple` | `boolean` | `false` | Enable multiple selection |
| `disabled` | `boolean` | - | Disable the select |
| `invalid` | `boolean` | - | Invalid state |
| `isItemEqualToValue` | `(item: TItem, value: TItem) => boolean` | - | Custom equality function for comparing items |

Plus all standard Select.Root props.
