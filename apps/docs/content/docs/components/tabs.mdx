## Overview

The Tabs component organizes content into multiple panels that can be toggled between. It's built on top of Base UI's Tabs component and provides an underline-style tab interface.

## Import

```tsx
import { Tabs } from "@px-ui/core";
```

## Props

### Tabs.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled active tab value |
| `defaultValue` | `string` | - | Uncontrolled default active tab value |
| `onValueChange` | `(value: string) => void` | - | Callback when active tab changes |
| `variant` | `"underline"` | `"underline"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |

### Tabs.List

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Tab triggers |

### Tabs.Trigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Tab value |
| `disabled` | `boolean` | - | Disable the tab |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Trigger content |

### Tabs.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Tab value (must match Trigger value) |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Tab panel content |

Plus all standard Base UI Tabs props.
