## Overview

The Spinner component displays an animated loading indicator to show that content or an action is in progress.

## Installation

```tsx
import { Spinner } from "@px-ui/core";
```

## Usage

### Basic Usage

```tsx
<Spinner />
```

### Sizes

```tsx
<Spinner size="small" />   // 16x16px
<Spinner size="medium" />  // 24x24px
<Spinner size="large" />   // 32x32px
```

### With Button

Commonly used in buttons during loading states:

```tsx
import { Button, Spinner } from "@px-ui/core";

<Button variant="primary" disabled>
  <Spinner size="small" />
  Saving...
</Button>
```

### Custom Color

The spinner inherits the current text color:

```tsx
<div className="text-ppx-primary-5">
  <Spinner />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"small" \| "medium" \| "large"` | `"small"` | Size of the spinner |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

The spinner includes:
- `role="status"` for screen readers
- `aria-label="Loading"` to describe the loading state

## Examples

### Loading Button

```tsx
import { useState } from "react";
import { Button, Spinner } from "@px-ui/core";

function SubmitButton() {
  const [loading, setLoading] = useState(false);

  const handleClick = async () => {
    setLoading(true);
    await saveData();
    setLoading(false);
  };

  return (
    <Button
      variant="primary"
      disabled={loading}
      onClick={handleClick}
    >
      {loading ? (
        <>
          <Spinner size="small" />
          Saving...
        </>
      ) : (
        "Save"
      )}
    </Button>
  );
}
```

### Full Page Loading

```tsx
import { Spinner } from "@px-ui/core";

function LoadingPage() {
  return (
    <div className="flex h-screen items-center justify-center">
      <div className="flex flex-col items-center gap-4">
        <Spinner size="large" />
        <p className="text-ppx-muted-foreground">Loading...</p>
      </div>
    </div>
  );
}
```

### Inline Loading

```tsx
import { Spinner } from "@px-ui/core";

<p className="flex items-center gap-2">
  <Spinner size="small" />
  Fetching data...
</p>
```
