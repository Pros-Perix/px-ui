title: Spinner
description: A loading spinner component for indicating async operations
---

## Overview

The Spinner component displays a rotating loading indicator. It's commonly used to show that an operation is in progress.

## Import

```tsx
import { Spinner } from "@px-ui/core";
```

## Usage

<Preview>
  <Spinner />
</Preview>

<CodeBlock>
```tsx
<Spinner />
```
</CodeBlock>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"small" \| "medium" \| "large"` | `"small"` | Size of the spinner |
| `className` | `string` | - | Additional CSS classes |
