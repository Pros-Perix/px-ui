## Overview

The Textarea component is a styled multi-line text input field that supports multiple sizes, validation states, and width variants. It's perfect for longer text input like descriptions or comments.

## Import

```tsx
import { Textarea } from "@px-ui/core";
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Size of the textarea |
| `widthVariant` | `"enforced" \| "full"` | `"full"` | Width behavior of the textarea |
| `invalid` | `boolean` | - | Whether the textarea is in an invalid state |
| `disabled` | `boolean` | - | Whether the textarea is disabled |
| `rows` | `number` | - | Number of visible text lines |
| `className` | `string` | - | Additional CSS classes |
| `placeholder` | `string` | - | Placeholder text |

Plus all standard HTML textarea attributes.
