## Overview

PX-UI Core provides a comprehensive set of components built with React and styled with Tailwind CSS. All components are designed with accessibility, performance, and developer experience in mind.

## Available Components

### Form Controls

- **[Button](/docs/components/button)** - Versatile button component with multiple variants and sizes
- **Input** - Text input fields with various states
- **Textarea** - Multi-line text input
- **Select** - Dropdown selection component
- **Checkbox** - Single or grouped checkboxes
- **Radio Group** - Radio button groups
- **Switch** - Toggle switch component

### Layout & Navigation

- **Breadcrumbs** - Navigation breadcrumb trail
- **Tabs** - Tabbed interface component
- **Menu** - Dropdown menu component
- **Separator** - Visual divider component

### Data Display

- **Avatar** - User avatar component
- **Avatar Group** - Grouped avatar display
- **Calendar** - Date selection calendar
- **Progress** - Progress indicator

### Overlays

- **Dialog** - Modal dialog component
- **Popover** - Popover overlay
- **Tooltip** - Contextual tooltip

### Feedback

- **Spinner** - Loading spinner component

### Advanced

- **Combobox** - Searchable select component
- **Date Picker** - Date selection with calendar
- **Block Checkbox Group** - Block-style checkbox selections
- **Block Radio Group** - Block-style radio selections

## Getting Started

All components can be imported from `@px-ui/core`:

```tsx
import { Button, Input, Select } from '@px-ui/core';
```

## Design Principles

- **Accessible**: Built with ARIA standards and keyboard navigation
- **Composable**: Components work together seamlessly
- **Customizable**: Easy to override styles with className
- **Type-Safe**: Full TypeScript support with proper types
- **Consistent**: Unified design language across all components
