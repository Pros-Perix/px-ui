# @px-ui/ai

AI components built on top of @px-ui/core for building chatbots, AI assistants, and other AI-powered interfaces.

## Features

- Built on @px-ui/core design system
- TypeScript support with full type inference
- Accessible components
- Customizable styling with Tailwind CSS

## Installation

```bash
pnpm add @px-ui/ai
```
